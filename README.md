# AI Chat Web App

This is a simple AI chat web application where users can send messages and receive AI-generated responses.

## Technologies Used
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express

## How to Run

1. Install Node.js
2. Open terminal in project folder
3. Run:
   npm install
4. Start server:
   node server.js
5. Open browser:
   http://localhost:3000

##  Environment Variables
Create a .env file and add:
OPENAI_API_KEY=your_api_key_here
